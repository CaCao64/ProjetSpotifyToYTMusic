ℹ️ Comment obtenir ces identifiants ?
  1. Rendez-vous sur le Developer Dashboard Spotify et connectez-vous.
  2. Cliquez sur Create App.
  3. Nommez votre application (ex: "SpotifyToYTMusic").
  4. Ajoutez l'URL de redirection suivante dans le champ Redirect URIs :
    http://127.0.0.1:3000/api/spotify/callback
  5. Cochez les conditions et enregistrez.
  6. Copiez le Client ID et le Client Secret (depuis les paramètres de l'app Spotify) et collez-les ici.

🔴 Configurer YouTube Music

ℹ️ Comment récupérer ces en-têtes ?
  1. Ouvrez une fenêtre de navigation privée.
  2. Allez sur music.youtube.com et connectez-vous à votre compte.
  3. Appuyez sur F12 pour ouvrir les outils développeur, puis allez dans l'onglet Réseau (Network).
  4. Filtrez les requêtes en tapant browse dans le filtre réseau.
  5. Rafraîchissez la page (F5). Cliquez sur une des requêtes browse qui a réussi (statut 200).
  6. Dans l'onglet En-têtes (Headers), faites défiler jusqu'à En-têtes de la requête (Request Headers).
  7. Sélectionnez et copiez tout le texte à partir de accept: */* jusqu'à la fin de la section des en-têtes, puis collez-le ici.
